# История промптов к AI-агенту (Cursor)

## Полный выгруз (все сообщения подряд)

Файл **[PROMPTS_ALL.md](./PROMPTS_ALL.md)** — **197** промптов в хронологически том виде, как они уходили агенту в основном чате разработки. Там же дословные логи ошибок из терминала и браузера, ссылки на фрагменты терминала (`@...terminals/...`), длинные уточнения по UI.

- Подряд идущие **дубликаты** одного и того же текста в транскрипте убраны (213 записей → 197 шагов).
- Строки, похожие на **API-ключи**, в файле заменены на плейсхолдер — в репозиторий секреты не попадают.

**Пересобрать файл** после новых сессий (если путь к `jsonl` стандартный для этого проекта на вашем ПК):

```bash
node scripts/build-prompts-all.mjs
```

При другом расположении транскрипта Cursor:

```bash
set TRANSCRIPT=C:\путь\к\файлу.jsonl
node scripts/build-prompts-all.mjs
```
